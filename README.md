# COD.js

Welcome! :) Try to enjoy first implementation of desing pattern MALEVICH

### Quick Start
```sh
npm i
npm run build
npm run prod
```

then try to visit
```
http://localhost:8888
```

### To develop

Run in first terminal window ( for npm live reload )
```sh
npm run develop
```
And run in the second ( for webpack auto ES2015 rebuild )
```sh
npm run watch
```
